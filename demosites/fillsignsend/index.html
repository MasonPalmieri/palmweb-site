Document workflow demo

Product-style demonstration of how a user creates an agreement, signs it, and sends it to a client — resulting in a completed, downloadable PDF.

  • ✓ Create an agreement from a PDF template
  • ✓ Upload your own PDF (no server upload)
  • ✓ Build an agreement from scratch (clause-based)
  • ✓ Sender signs first (Fill → Sign → Send)
  • ✓ Completed document becomes available for download

This demo is static and runs entirely in the browser using localStorage. Nothing is sent to a server.
